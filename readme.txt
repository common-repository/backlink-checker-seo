== Installation ==
From your WordPress dashboard

1. Visit 'Plugins > Add New'

2. Search for keyword 'Backlink Checker SEO'

3. Install the Plugin, then Activate it.

4. Find plugin at the bottom of the left sidebar and Click on "Backlink Checker SEO" in Admin menu.

5. Input the Demo API Key, site URL to test out the report.
